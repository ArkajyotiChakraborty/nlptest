## Swap Cohyponyms

This test uses wordnet to replace words with their cohyponyms.

**alias_name:** `swap_cohyponyms`

</div><div class="h3-box" markdown="1">

#### Config
```yaml
swap_cohyponyms:
    min_pass_rate: 0.6
```
- **min_pass_rate (float):** Minimum pass rate to pass the test.

</div><div class="h3-box" markdown="1">
